# AI Algorithms in Python 🧠


This repository contains implementations of classic Artificial Intelligence and Machine Learning algorithms written from scratch in Python.

I originally started this project to better understand how these algorithms work internally instead of only using high-level libraries like `scikit-learn` or `TensorFlow`. Most of the projects here were built as learning experiments and gradually improved over time.

---

## Project Structure

- **Neural Networks**
  Multi-layer Perceptron (MLP) implementation trained on the MNIST dataset.

- **RBF Networks**
  Function approximation using Radial Basis Function networks.

- **Decision Trees**
  Decision tree implementation from scratch for classification tasks.

- **Fuzzy Systems**
  Fuzzy logic controller for computation offloading, compared with local and random approaches.

- **Hopfield & Kohonen (SOM)**
  - Hopfield network for associative memory using dummy data
  - Self-Organizing Maps (SOM) for GloVe word embedding visualization

- **Othello AI**
  Strategic gameplay agent using the Minimax algorithm.

## How to Use

Clone the repository and run the project you are interested in:
```bash
git clone https://github.com/setayeshbaghaee/AI-Algorithms-python.git
cd AI-Algorithms-python

# Navigate to a specific project directory
python <script_name>.py
```
